Extensões Uteis para o Visual Studio Code.

Nesse guia vamos te apresentar algumas extensões uteis para o VSCode. Todas as entensões aprensetadas aqui foram testadas e são recomenadas para uso.

Como instalar extensões?

  1. Abra o Visual Studio Code e pressione Ctrl + Shift + P
  2. Busque por: Install Extensions
  3. Selecione a extensãõ que deseja e prencione em Instalar

Plugins

  • Dracula Theme Um dos mais famosos temas, suportados em diversas IDE’s e gratis.

  • Bracket Pair Colorizer Essa extensão permite que os colchetes correspondentes sejam identificados com cores. O usuário pode definir quais caracteres combinar e quais cores usar.

  • Auto Close Tags Adicione automaticamente a tag de fechamento HTML / XML. Combine com as funções de Auto Complete Tags e Auto Rename Tags

  • Color Highlight Esta extensão estiliza as cores css/web encontradas no seu documento.

  • Material Icon Theme O Tema dos ícones de materiais fornece muitos ícones baseados no Material Design para código do Visual Studio.

  • Prettier + ESLint + EditorConfig Esses são realmente importantes, por isso temos um posto só deles. Acesse o link: http://pluggedcomputing.gitlab.io/post/style_guide e veja como instalar e configurar essas extensões para trabalharem para você.

  • Markdown All in One Plugin para escrever e ler Markdown dentro do VSCode, muito útil para documentações o README’s do Github e GitLab;

  • ES7 React/Redux/GraphQL/React-Native snippets Esta extensão fornece trechos de JavaScript e React / Redux no ES7 com recursos de plug-in Babel para VS Code

  • Rocketseat React Native Snippets Este projeto visa a disponibilização de um conjunto de Snippets ou atalhos para criação de componentes e arquivos de configuração em aplicações React Native.

  • Rocketseat React JS Snippets Este projeto visa a disponibilização de um conjunto de Snippets ou atalhos para criação de componentes e arquivos de configuração em aplicações ReactJS.
